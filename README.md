# Creatine Supplementation Simulation & EDA

This project explores a messy **synthetic dataset** designed to simulate the hypothesis that higher creatine dosages may be associated with improvements in:

- Mood  
- Sleep quality  
- Cognitive performance  

> ⚠️ Note: This is a simulated dataset for educational and exploratory analysis only. It does not represent real clinical findings.

---

## 📊 Dataset Overview

The dataset includes the following variables:

- Name  
- Age  
- Sex  
- Mood score  
- Sleep score  
- Cognition score  
- Creatine dosage  

---

## 🔍 Analysis Performed

After generating the noisy dataset with Chatgpt & Gemini, I performed data cleaning and Exploratory Data Analysis (EDA), including:

- Correlation matrix analysis  
- Scatter plots to observe relationships between variables  
- Violin plots for distribution comparisons  

---

## 📈 Tools & Libraries Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
